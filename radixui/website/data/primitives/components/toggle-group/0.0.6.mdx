# Toggle Group

<Description>
  A set of two-state buttons that can be toggled on or off.
</Description>

<Highlights
  features={[
    'Full keyboard navigation.',
    'Support single and multiple pressed buttons.',
    'Can be controlled or uncontrolled.',
  ]}
/>

## Installation

Install the component from your command line.

```bash
npm install @radix-ui/react-toggle-group
```

## Anatomy

Import the component.

```jsx
import * as ToggleGroup from '@radix-ui/react-toggle-group';

export default () => (
  <ToggleGroup.Root>
    <ToggleGroup.Item />
  </ToggleGroup.Root>
);
```

## Basic example

Create your styled toggle group component from the primitive parts.

```jsx
import { styled } from 'path-to/stitches.config';
import * as ToggleGroup from '@radix-ui/react-toggle-group';

const StyledItem = styled(ToggleGroup.Item, {
  appearance: 'none',
  backgroundColor: 'transparent',
  border: 'none',
  padding: '5px 10px',
  margin: '0 1px',
  boxShadow: 'inset 0 0 0 1px gainsboro',
  overflow: 'hidden',
  borderRadius: 3,

  '&:focus': {
    outline: 'none',
    boxShadow: 'inset 0 0 0 1px dodgerblue, 0 0 0 1px dodgerblue',
  },

  '&[data-state=on]': {
    backgroundColor: 'gainsboro',
  },
});

export default () => (
  <ToggleGroup.Root type="single">
    <StyledItem value="left">Left</StyledItem>
    <StyledItem value="center">Center</StyledItem>
    <StyledItem value="right">Right</StyledItem>
  </ToggleGroup.Root>
);
```

<Note>

**Note:** This example uses [Stitches](https://stitches.dev) but any styling solution is [compatible](/docs/primitives/overview/styling).

</Note>

## Accessibility
