# Color contrast tests

<Description>
  A test suite for testing color contrast in accordance with WCAG guidelines.
</Description>

## AA Normal Text

Step `11` is designed for `400` weight text in any size, and guaranteed to work well on a step `3` background. Each test should pass 4.5:1 contrast ratio.

## AA Large Text

Step `9` step is designed for solid backgrounds, and guaranteed to work well behind white text in `600` weight at 19px. Each test should pass 3:1 contrast ratio.
